9:45-10:00 Recepção dos participantes
10:00-10:45 PowerShell DSC, A New Approach in Deploying SharePoint Environments
Rodrigo Pinto (SharePoint PT)

One of the fastest-moving technologies today is PowerShell Desired State Configuration. It literally transforms how Implementers deploy and manage on-premises resources and those extended to the cloud.

Desired State Configuration is technology for declarative configuration of systems.

Oversimplifying a bit, DSC lets you create specially formatted text files that describe how a system should be configured. By deploying those files to the actual systems, they magically configure themselves.

Get a solid foundation in how to provision a SharePoint farm so that you can deploy it to different environments.
10:45-11:00 Intervalo
11:00-11:45 Azure SQL Database - Elastic Database
Vitor Faria Tomaz (NetPonto/AzurePT)
 

Scaling up or down in SQL Database it’s easy but if we need more than one SQL Database can handle (in terms of storage or performance) we need to scale out (and scale in). Adding elasticity to the database tier can be complex.

Azure SQL Database Elastic Scale adds both developer and management functionality to our highly scalable applications.

In this session we will see how Azure DB Elastic Scale implements the infrastructure aspects of sharding by taking a look at the set of .Net libraries and Azure service templates which allow us to focus on the business logic of the application instead of custom sharding implementations.
